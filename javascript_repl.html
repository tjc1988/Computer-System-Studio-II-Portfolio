JavaScript repl programs

This is a list of JavaScript programs that I made using repl in the Programming 1 course
in the Spring 2020 semester. The list starts with the most recent to the oldest.



Website example
Website example (version 2)

This was for an assignment where we were given a wireframe for a webpage and we had to
make a webpage to its specifications. I didn't get it exactly like the wireframe but it was
very close. Later I found some of the errors and fixed them in version 2.



Address form program

This program has a form that you fill out that would be similar to an address form on a
website when making an order. The main objective for this assignment was to program it so that
when you press the submit button, it will display the address that you entered in the correct
format that you would typically see an address written.



Swimming pool volume calulation

This is a HTML file that uses JavaScript to calculate the volume of a pool with preset number values.
